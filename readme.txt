WPBooking is an All-in-one Booking System

== Description ==

WPBooking is an All-in-one Booking System that help you create a booking system is very easy

== Installation ==

1. Upload the plugin zip file to the `/wp-content/plugins/` directory and unzip it, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WPBooking-> Settings screen to configure the plugin, include: Page, Currency, Email, Review Options, Payment Options
